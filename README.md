# Immich-Go Docker Image

This repository contains the Docker image for [immich-go](https://github.com/simulot/immich-go), a tool for managing and processing images.

## Docker Hub Repository

The Docker image is available on Docker Hub: [johnnmackk/immich-go](https://hub.docker.com/repository/docker/johnnmackk/immich-go/general)

## Usage

To pull the Docker image, use the following command:

```bash
docker pull johnnmackk/immich-go:latest
```

To run the Docker image, execute:

```bash
docker run --rm -v /path/to/your/data:/data johnnmackk/immich-go
```

Replace `/path/to/your/data` with the path to your data directory.
